# Analizador sintáctico con interfaz gráfica

Este proyecto consiste en un analizador léxico y sintáctico desarrollado en Python utilizando PLY (Lex/Yacc), capaz de procesar estructuras tipo JSON y generar una salida en HTML.

## 🚀 Funcionalidades
- Análisis léxico mediante expresiones regulares
- Análisis sintáctico con reglas gramaticales
- Validación de estructuras de datos
- Generación automática de HTML
- Interfaz gráfica con Tkinter
- Carga de archivos mediante drag & drop
- Visualización de errores

## 🧠 Tecnologías utilizadas
- Python
- PLY (lex / yacc)
- Tkinter

## ▶️ Cómo ejecutar

1. Instalar dependencias:
   pip install ply tkinterdnd2
2. Ejecutar el programa:
   python interfaz.py
3. Cargar un archivo JSON o TXT desde la interfaz

## 📁 Estructura del proyecto
- `analizador_lexico.py` → define tokens
- `parser.py` → define gramática
- `interfaz.py` → interfaz gráfica
