# Analizador sintáctico con interfaz gráfica

Proyecto desarrollado en Python utilizando PLY (Lex/Yacc) para el análisis de estructuras tipo JSON y generación de HTML.

## Funcionalidades
- Análisis léxico y sintáctico
- Validación de estructura de datos
- Generación automática de HTML
- Interfaz gráfica con Tkinter
- Carga de archivos mediante drag & drop
- Visualización de errores en tiempo real

## Tecnologías
- Python
- PLY (lex/yacc)
- Tkinter

## Uso
1. Ejecutar interfaz.py
2. Cargar un archivo .json o .txt
3. Procesar archivo
4. Se genera un archivo HTML automáticamente
